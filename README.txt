Ce programme permet de copier des chaines de caractères dans le presse papier en fonction de raccourcies clavier. 
Le code actuel permet d'accèder aux symboles liés à la touche alt-gr dans le cas ou elle serait défectueuse. 

Pour utiliser ce programme vous avez besoin d'installer les modules keyboard et pyperclip
>pip install keyboard
>pip install pyperclip

Vous pouvez lancer le programme depuis un IDE ou l'invite de commande cmd.
Vous pouvez également l'utiliser en exécutable.
